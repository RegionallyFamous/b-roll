WP Desktop Mode is the computer

ODD is the fun weird stuff on top.

WP Desktop Mode already makes WordPress feel like a tiny operating system: windows, a dock, shortcuts, and apps. ODD decorates that desktop with animated wallpaper worlds, full-color icon sets, a seven-app catalog that installs straight into WordPress, and a control panel with personality.

WP Desktop Mode gives WordPress a shell

Instead of another admin page, you get a real desktop metaphor: launchers, windows, shortcuts, and a dock that makes WordPress feel like software you can inhabit.

ODD makes the shell feel alive

Animated wallpaper scenes, reactive Iris moments, and wild desktop moods turn the calm WP Desktop foundation into a strange little place with movement and attitude.

A one-click app drawer, bundled in

ODD ships seven installable apps — a kanban board, a pixel editor, a pomodoro, a color picker, an invoicing tool, a team wiki, and a real Web Audio synthesizer — each opening in its own WP Desktop window.

The ODD app drawer

Seven apps that ride along with the plugin.

Every ODD install ships a Bazaar-compatible catalog. One click installs an app into WordPress; it opens in its own desktop window, icon and all. No admin page detours, no extra plugins.

Mosaic

Pixel editor with drawing tools, fills, undo/redo, and PNG export when your 32×32 masterpiece is ready to ship.

creativeeditor

Flow

Pomodoro with ambient soundscapes, per-session task lists, and a heatmap showing exactly which days you crushed it.

productivityfocus

Board

Cards. Columns. Drag. Drop. Done. A no-nonsense project board that shows what's moving, what's stuck, and what's finally done.
