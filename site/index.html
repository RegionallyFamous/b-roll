WP Desktop Mode gives WordPress a shell

Instead of another admin page, you get a real desktop metaphor: launchers, windows, shortcuts, and a dock that makes WordPress feel like software you can inhabit.

ODD makes the shell feel alive

Animated wallpaper scenes, reactive Iris moments, and wild desktop moods turn the calm WP Desktop foundation into a strange little place with movement and attitude.

Apps and icons complete the illusion

ODD reskins the dock, ships matching app icons, and lets installable apps open in desktop windows so the whole admin starts to feel like its own toy operating system.
